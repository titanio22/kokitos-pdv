💻 PDV Desktop para Windows

📦 Baixar PDV Desktop

Versão para Windows — receba pedidos direto no seu computador sem precisar abrir o navegador.

📋 Como instalar

1
Baixe o arquivo ZIP clicando no botão acima.
2
Extraia o conteúdo para uma pasta (ex: C:\PDV-Kokitos).
3
Execute o arquivo iniciar-pdv.bat como Administrador na primeira vez.
4
Pronto! O PDV vai abrir em uma janela própria 🎉

✅ Requisitos

  • Windows 10 ou superior
  • Node.js 18+ instalado
  • Impressora térmica configurada (opcional)

Não tem Node.js? Baixe aqui e instale.

🔧 Funcionalidades

  • 🪟 Janela nativa do Windows — sem precisar de navegador
  • 🖨️ Impressão direta na impressora do Windows
  • 🔔 Notificações no sistema quando chegar pedido novo
  • 🗨️ Minimiza para a bandeja do sistema
  • 🔄 Auto-atualização dos pedidos a cada 5 segundos
